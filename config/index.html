Configuration¶

This module can be configured by editing the configuration file. This file is installed in /etc/opt/dans.knaw.nl/dd-transfer-to-vault/config.yml when using the RPM. The settings are explained with comments in the file itself. An on-line version of the latest configuration file can be found here.
